Hostels in Ranchi

Web Application

Project Overview

Hostels in Ranchi is a web-based application designed to simplify the hostel search process for students, interns, and working professionals relocating to Ranchi. The platform provides hostel listings with important information such as facilities, room availability, pricing, and contact details. Instead of visiting multiple hostels physically, users can compare options from a single platform before making an enquiry. This project demonstrates practical implementation of frontend and backend web development concepts through the integration of HTML, CSS, JavaScript, PHP, and MySQL.

Problem Statement

Finding suitable hostel accommodation is often a time-consuming process because information is scattered across different websites and local sources. This application was developed to centralize hostel information into one easy-to-use platform, helping users quickly compare available options and contact hostel owners without unnecessary effort.

Objectives

  • Develop a centralized hostel information platform.
  • Provide a simple and responsive user interface.
  • Enable users to view hostel facilities and pricing.
  • Implement database connectivity using MySQL.
  • Ensure reliable user interaction through form validation.
  • Apply full-stack web development concepts.

Key Features

  • Hostel Listing System
  • Detailed Hostel Information
  • Contact & Enquiry Form
  • Responsive Web Design
  • JavaScript Form Validation
  • PHP Backend Processing
  • MySQL Database Connectivity
  • Local Deployment using XAMPP/WAMP

Technology Stack

Frontend: HTML5, CSS3, JavaScript

Backend: PHP

Database: MySQL

Development Environment: XAMPP / WAMP Server

My Contributions

I was responsible for designing the complete frontend interface using HTML and CSS while ensuring responsiveness across desktop and mobile devices. I integrated PHP with MySQL to establish database connectivity, implemented JavaScript-based form validation to improve user experience, and tested the application locally using XAMPP. Throughout the project, I focused on writing clean code, debugging issues, and creating an intuitive user interface that allows users to access hostel information efficiently.

Skills Gained

This project strengthened my understanding of full-stack web development, relational databases, CRUD operations, frontend- backend integration, responsive web design, software testing, debugging, and project deployment in a local development environment.

Future Enhancements

MNC JobHub

AI-Powered Job Search & Career Platform (In Progress)

Project Vision

MNC JobHub is a full-stack web application currently under development with the objective of simplifying the job search process for freshers and experienced professionals. Instead of visiting multiple company career portals individually, users will be able to search opportunities from various multinational companies through a single platform. The project is being developed using the MERN stack and focuses on delivering a scalable, responsive, and user-friendly experience while demonstrating modern full-stack development practices.

Problem Statement

Job seekers often spend significant time searching across multiple company career portals. Each organization follows a different website structure, making it difficult to track opportunities efficiently. MNC JobHub aims to centralize publicly available job listings by providing advanced search and filtering capabilities, helping users discover relevant opportunities more quickly while redirecting them to the official company career page for applications.

Project Objectives

  • Create a centralized job search platform.
  • Allow searching by job role, company, location, and experience.
  • Display detailed job descriptions.
  • Provide direct links to official company career portals.
  • Develop a secure user authentication system.
  • Build a responsive and scalable web application.
  • Apply full-stack development concepts using the MERN stack.
